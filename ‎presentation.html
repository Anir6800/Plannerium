FRONTEND TECH

  • Vanilla JS (ES6+): No heavy frameworks. Pure performance.
  • CSS Variables: Dynamic theming (Dark/Light/Cyber).
  • Glassmorphism: Modern, layered UI design.
  • SPA Architecture: Seamless transitions without reloads.

BACKEND CORE

  • Python Flask: Lightweight, robust API server.
  • Google Gemini 2.5 Flash: The intelligence engine.
  • Prompt Engineering: Structured JSON outputs for reliability.

DATABASE & SYNC

Firebase Realtime Database

  • Instant updates across devices.
  • User profile and XP persistence.
  • JSON Structure: `users/{uid}/projects/{pid}`

GAMIFICATION ENGINE

XP SYSTEM

Task Completion = +50 XP

Plan Generation = +100 XP

SECURITY PROTOCOLS

  • Authentication: Firebase Auth (Email/Password & Google).
  • Environment Variables: API Keys protected via `.env`.
  • Data Isolation: Users can only access their own node path.

USER EXPERIENCE & DESIGN
